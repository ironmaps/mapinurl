Why mapinurl?

Many a times, I've wished to share some location data with friends/family and other tools on the internet are too extravagant for tiny personal usecases.

With "mapinurl", your data stays with you (literally in the URL), and eventually with people and machines you share this URL with.

You can use it to generate small maps to embed in class-notes, maps to share weekend hiking trails with friends, etc.

Currently it supports only data generated in the webpage, since it's not meant to be used to work with a 100MB GeoJSON. :) But I'll add some support soon.

Of course, like everything this has its own drawbacks. Since the link contains the actual data, they are very long.

How to use?

  • Draw shapes, and add notes to shapes by clicking on them.
  • Use the toolbar on the left to select shapes while drawing
  • Edit tooltip notes(doesn't require a manual save from toolbar). Edit/delete geometries.
  • The link displayed in the bottom and Copy link to Map button can be used to get a shareable link to this map.
